> Select which input drives an output.

![[Data Selector 2024-09-23 09.38.49.excalidraw]]

Using [[Entered Variables|entered variables]]:

| S1  | S2  | Y   |
| --- | --- | --- |
| 0   | 0   | I0  |
| 0   | 1   | I1  |
| 1   | 0   | I2  |
| 1   | 1   | I3  |

There are $\log_2(N)$ selectors, where $N$ is the number of inputs.
